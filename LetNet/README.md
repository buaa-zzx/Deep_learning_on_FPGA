# 手写数字识别加速
 该项目在2019年的DAC-SDC大赛中取得了冠军，该项目在Ultra96系列FPGA上实现了 0.716 IoU和25.05 FPS，性能优异。本项目将该项目在百度Edgeboard ZU3系列FPGA上进行了复现。  

 # 开发工具  
vivado2018.3
 
# 文件说明  
pytorch: 神经网络实现
vivado：FPGA硬件工程   
HLS：加速IP实现工程  
pynq：上传至开发板的文件
